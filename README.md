# Project Name
Parallel Paths : Unseen Echos of Introverted Souls

## Project Description

The overall concept and theme of the video revolve around the intersection of these introverted souls that both share the same interests but will not interact with one another, and the goal is to adress the ongoing problem of introverted souls having issues with interacting with potential friends but also normalizing introvertion at the same time.

However, after reviewing the clips, I had the thought of having it adress and normalize something that is seen as an issue but with a fun tone to it. Hving prior experience playing the ukulele, I decided to cover my favorite song.
