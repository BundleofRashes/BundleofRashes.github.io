# Project Name
Parallel Paths : Unseen Echos of Introverted Souls

## Project Description

The overall concept and theme of the video revolve around the intersection of these introverted souls that both share the same interests but will not interact with one another, and the goal is to address the ongoing problem of introverted souls having issues with interacting with potential friends but also normalizing introversion at the same time.

## Process

### Development
- Since we were only given 30 minutes to film the video, our initial concept revolved around the stop motion theme. However, after reviewing the clips, I decided to have it addressed and normalize something seen as an issue but with a fun tone. Having prior experience playing the ukulele, I decided to cover my favorite song.

### Design
- When it came to the color palette, I could not choose one, so I chose two: Night and day. I did not want it to be one color and decided to search for images. After searching, I found art from my favorite show and asked if I could use it.

### Code
- This is my first time coding using HTML and CSS, and this wasn't easy. I ran into many difficulties building this website. However, after asking friends with previous coding experience, they helped me navigate and debug my errors.

## Reflection/Evaluation

### Lessons Learned
- I am happy with how this turned out overall. It's nothing like I expected, but it is to a satisfactory level.

### Future Improvements
- I would like to take this further and experiment with things I have not touched upon yet.
